Introduction to Lua

Lua is a lightweight, powerful, and easy-to-learn scripting language widely used in Roblox game development. Below is a comprehensive list of Lua functions, structures, and code examples to help you understand and master Lua for Roblox development.

Lua Functions with Examples

  • print(): Outputs text to the console.
    Example: 
    
print("Hello, World!")
  • pairs(): Iterates through a table in no particular order.
    Example: 
    
local fruits = {apple = "red", banana = "yellow", grape = "purple"}
for key, value in pairs(fruits) do
    print(key .. ": " .. value)
end
  • ipairs(): Iterates through an array in order.
    Example: 
    
local numbers = {10, 20, 30}
for i, v in ipairs(numbers) do
    print(i .. ": " .. v)
end
  • tonumber(): Converts a value to a number.
    Example: 
    
local num = tonumber("10")
print(num + 5) -- Outputs 15
  • tostring(): Converts a value to a string.
    Example: 
    
local str = tostring(123)
print("The number is: " .. str)
  • math.random(): Generates a random number.
    Example: 
    
local randomNum = math.random(1, 100)
print(randomNum)
  • table.insert(): Inserts a value into a table.
    Example: 
    
local myTable = {10, 20, 30}
table.insert(myTable, 40)
print(myTable[4]) -- Outputs 40
  • table.remove(): Removes a value from a table.
    Example: 
    
local myTable = {10, 20, 30}
table.remove(myTable, 2)
print(myTable[2]) -- Outputs 30
  • string.len(): Returns the length of a string.
    Example: 
    
local myString = "Hello, World!"
local length = string.len(myString)
print(length) -- Outputs 13
  • string.sub(): Returns a substring of a string.
    Example: 
    
local myString = "Hello, World!"
local subString = string.sub(myString, 1, 5)
print(subString) -- Outputs "Hello"

Lua Operators

Lua supports the following operators:

  • Addition (+)
    Example: 
    
local sum = 10 + 20
print(sum) -- Outputs 30
  • Subtraction (-)
    Example: 
    
local difference = 20 - 10
print(difference) -- Outputs 10
  • Multiplication (*)
    Example: 
    
local product = 10 * 5
print(product) -- Outputs 50
  • Division (/)
    Example: 
    
local quotient = 50 / 10
print(quotient) -- Outputs 5
  • Concatenation (..)
    Example: 
    
local message = "Hello, " .. "World!"
print(message) -- Outputs "Hello, World!"

Control Structures

Lua provides the following control structures:

  • If-else statements
    Example: 
    
local number = 10
if number == 10 then
    print("The number is 10.")
else
    print("The number is not 10.")
end
  • While loops
    Example: 
    
local counter = 1
while counter <= 5 do
    print(counter)
    counter = counter + 1
end
  • For loops
    Example: 
    
for i = 1, 5 do
    print(i)
end

Tables

Tables are the primary data structure in Lua, used to create arrays, dictionaries, and more. Here's an example of creating and manipulating tables:


-- Create a table (array)
local fruits = {"apple", "banana", "grape"}

-- Access elements
print(fruits[1]) -- Outputs "apple"

-- Modify elements
fruits[2] = "orange"
print(fruits[2]) -- Outputs "orange"

-- Insert new element
table.insert(fruits, "kiwi")
print(fruits[4]) -- Outputs "kiwi"

Functions

Functions are first-class values in Lua, which means they can be stored in variables, passed as arguments, and returned from other functions.


-- Define a function
function greet(name)
    print("Hello, " .. name)
end

-- Call the function
greet("Charley") -- Outputs "Hello, Charley"

Metatables and Metamethods

Metatables allow you to change the behavior of tables in Lua. Metamethods are special functions that handle certain operations.


local myTable = {}

local meta = {
    __index = function(table, key)
        print(key .. " not found!")
    end
}

setmetatable(myTable, meta)

-- Access a non-existing key
print(myTable.someKey) -- Outputs "someKey not found!"
